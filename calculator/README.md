

# 📟 Calculator App

Welcome to the **Calculator App**! This is a simple, command-line based calculator written in C. It supports basic arithmetic operations like addition, subtraction, multiplication, division, and more.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Remainder
- 🔋 Power calculation
- 🚪 Exit option
